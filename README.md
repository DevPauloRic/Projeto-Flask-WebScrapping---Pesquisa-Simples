# Micro-Dashboard de Word Cloud

## Descrição
Este projeto foi desenvolvido como parte de uma aula prática na faculdade, com o objetivo de aprender a construir aplicações web usando **Flask**, fazer **web scraping** e apresentar os dados de forma visual por meio de gráficos.

O projeto permite que o usuário realize uma busca por qualquer termo de sua escolha e, a partir dos resultados coletados, gere uma **Word Cloud** mostrando as palavras mais frequentes encontradas.

## Funcionalidades
- Interface web simples utilizando Flask.  
- Permite ao usuário buscar qualquer termo.  
- Realiza scraping de resultados online relacionados ao termo pesquisado.  
- Processa e analisa o texto coletado para identificar palavras mais frequentes.  
- Gera um gráfico de **Word Cloud** interativo mostrando as palavras mais comuns.  
- Fornece insights visuais imediatos sobre o conteúdo pesquisado.

## Tecnologias Utilizadas
- **Python**: linguagem principal do projeto.  
- **Flask**: framework web para criação do dashboard.  
- **BeautifulSoup / Requests**: para web scraping.  
- **Matplotlib / WordCloud**: para geração de gráficos.  
- **HTML/CSS**: para estilização da interface.

## Objetivo de Aprendizado
- Praticar **web scraping** e manipulação de dados.  
- Aprender a integrar Python com uma aplicação web.  
- Visualizar dados de forma interativa e compreensível para o usuário.  
- Explorar análise de texto e frequência de palavras em um contexto real.
